# Notes Directory

这个目录存放已发生工作的记录、复盘和阶段性结论，重点是“这轮到底做了什么、遇到了什么问题、怎么验证的”。

## 文件说明

- `2026-03-21-rag.md`
  早期 RAG 基础链路与阶段状态记录。
- `2026-03-22-contextual-retrieval-observability.md`
  contextual retrieval / observability 方向的思考与记录。
- `2026-03-22-query-planning.md`
  query planning 演进记录。
- `2026-03-22-routing-rerank.md`
  routing 与 rerank 的中间阶段记录。
- `2026-03-24-cloud-rerank-rollout.md`
  云端 rerank 接入与回归结果记录。
- `2026-03-24-query-hardcase-closure.md`
  query hard-case 收口与专项验证记录。
- `2026-03-28-session-workspace-lifecycle.md`
  `web-game` 会话工作台第一轮生命周期能力记录，包括最近会话恢复、重命名、归档/恢复和 memory timeline 初版。
- `recall-accuracy-optimization.md`
  召回准确率相关的长期笔记。

## 使用建议

- 想知道“为什么这一轮会做某个改动”，优先看对应日期的 note
- 想判断“这轮到底算不算收口”，看 note 里的 `Current Judgment`
- 想接着做新的范围，再配合 `guidance/` 里的下一步计划一起看
